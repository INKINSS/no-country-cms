# src/hooks

Este directorio contiene hooks de React reutilizables a nivel aplicacion.

Ejemplos para este CRM:

- Hooks de consulta a APIs (TanStack Query).
- Hooks de estado compartido entre vistas.
- Hooks utilitarios de paginacion, debounce y manejo de filtros.

Si un hook solo aplica a una feature, ubicalo en src/features/<feature>/hooks.
